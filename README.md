# fde-education

Personal self-education repo for learning LLM primitives.

Starting point:
- `llm-primitives/` for terminal-first experiments with observable LLM chains
- local env setup for API keys and related config

Working principle for this repo:
- use terminal-first flows unless there is a strong reason not to
- keep environment variables local
- make model activity observable through explicit step-by-step output

This is intentionally lightweight and meant to grow over time as new examples and notes are added.
